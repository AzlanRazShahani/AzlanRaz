## 01 · About Me

<img align="right" src="https://user-images.githubusercontent.com/74038190/213910845-af37a709-8995-40d6-be59-724526e3c3d7.gif" width="320" />

I'm **Azlan Raz**, a student developer based in **Pakistan**, currently laying down the fundamentals of programming and web development. My focus right now is **HTML, CSS, and Python**, paired with growing comfort in the **Linux command line** and version control through **Git & GitHub**.

Development isn't my only craft — I also design **logos, banners, and flyers** in **Canva**, which gives me a visual, design-conscious lens on the projects I build. Lately, I've started exploring **AI prompting**, curious about how thoughtful, structured prompts can become a real part of a developer's workflow.

This README isn't a highlight reel of finished achievements — it's an honest, evolving snapshot of where I am and where I'm headed.
